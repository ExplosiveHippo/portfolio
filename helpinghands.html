Helping Hands Mobile App Mockup

Description: This project was a solo project for my Design of GUI class at RIT. We were to create a mockup for an app of our choice. I decided to create an application aimed at allowing parents to teach cooking skills to their children. The goal of the project was to use cooking skills as a way of teaching independence as well as basic math skills like measuring.

Tech: The design was created in Photoshop.
