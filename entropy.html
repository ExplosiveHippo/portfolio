Entropy Image Analyser

Description: This project is still in development with two of my classmate, Chris Knepper and Dustin Raimondi. Entropy is a command line tool that allows a user to supply an image and be given a score based on whether the image would be a good wallpaper for their system. The tool is fairly simple right now but will be completed shortly.

Tech: Entropy uses Python as it's core language and utilized the Pillow library.
