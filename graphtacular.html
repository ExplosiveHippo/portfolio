Graphtacular.js

Description: Graphtacular.js is a JavaScript graphing library developed with my classmate Mike Nolan. Graphtacular was designed to be light-weight and easy to implement for even novice JavaScript Developers. It's still in development and we hope to continue to add new functionality and additional graph types. You can view a presentation about Graphtacular here

Tech: Graphtacular is written in object oriented Javascript and utilized HTML5 Canvas.
