Fractionauts

Description: Fractionaunts is a Python based game designed to run on the One Laptop One Child XO Laptops. The purpose of Fractionaunts is to teach 4th grade students about fraction operations. Fractionaunts is published on SugarLabs and available to download. Fractionauts was created with five other students: Mike Nolan, Chris Knepper, Valerie Magri, Brendan George and John Savage.

Tech: Fractionauts is built in Python and used the Pygame Library.
